# Criando um novo repositório
--- git init ---

# Adicionando tudo

# Adicionando um comentário, para controle de desenvolvimento
--- git commit -m "comentários das alterações"  ---

# Realizando o upload das alterações
--- git push origin master  ---

# Atualizar a pasta 
--- git pull ---
